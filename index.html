Sete Projetos JavaScript em 7 DIAS


Por André Augusto Eigenheer

Descrição do projeto:

Nesse projeto temos 7 desafios em JavaScript ministrada pela B7Web através do Prof. Bonieky Lacerda

Projetos:

Neste primeiro projeto realizei a criação de uma bateria (instrumento)
No segundo projeto realizei a criação de dois relogios, um digital e um analógico
Já no terceiro projeto realizei a criação de uma API de verificação de temperatura
Agora no quarto projeto realizei a criação de um clássico dos jogos! Um jogo da velha
Neste quinto projeto realizei a criação de um dashboard simular ao canvas
Já no sexto projeto realizei a criação de um quiz
Já no sétimo projeto realizei a criação de uma API de verificação de temperatura

Tecnologias utilizadas:

JavaScripit

Html

CSS
